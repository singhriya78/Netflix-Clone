# Netflix India — Landing Page Clone

A responsive recreation of Netflix India's landing page, built from scratch using pure HTML and CSS — no frameworks, no JavaScript, no libraries beyond Google Fonts and Font Awesome icons.

**Live Demo:** [netflix-clone-singhriya78s-projects.vercel.app](https://netflix-clone-singhriya78s-projects.vercel.app/)

>**Disclaimer:** This is a non-functional student project built purely for learning frontend web development. It is **not affiliated with, endorsed by, or connected to Netflix, Inc.** in any way. No user data is collected, stored, or transmitted — form inputs are for UI demonstration only. "NETFLIX" is a registered trademark of Netflix, Inc.

---

## Features

- Fixed navbar with transparent gradient background on scroll
- Language selector and Sign In button
- Full-viewport hero section with background image and email CTA
- Curved, glowing partition divider between hero and content
- "Trending Now" horizontally scrollable card row with hover animations
- "More Reasons to Join" section with custom SVG icons
- Fully styled FAQ accordion section
- Footer with email input, "Get Started" button, and a 3-column link layout
- **Fully responsive** — adapted for tablets and mobile devices
- Google Fonts — Bebas Neue + Martel Sans
- Font Awesome 7 icons throughout

## Built With

- HTML5
- CSS3 (Flexbox, media queries, CSS Grid)
- Font Awesome 7
- Google Fonts
- Deployed on [Vercel](https://vercel.com/)

## How to Run Locally

No installation or build step needed.

1. Clone the repo
   ```bash
   git clone https://github.com/singhriya78/Netflix-Clone.git
   ```
2. Open `index.html` directly in your browser — that's it.

## Responsive Design

The layout adjusts across three breakpoints (desktop, tablet ≤768px, and mobile ≤420px), with adaptive font sizes, stacked buttons/inputs, a horizontally scrollable trending row, and a collapsing footer link grid for smaller screens.

## What I Learned

- Flexbox layout patterns for navbars, hero sections, and card rows

- Building accessible, responsive forms without JavaScript
- Structuring and debugging CSS across breakpoints
