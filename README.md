# Netflix India Clone 

A pixel-perfect(ish) clone of Netflix India's landing page — 
built from scratch using only HTML and CSS. No frameworks, 
no libraries, just pure code and a lot of patience. 

## Features

- Fixed navbar with transparent gradient background
- Language selector and Sign In button
- Hero section with full background image
- Curved glowing partition divider
- Trending Now section with hover effects
- More Reasons to Join section with custom SVG icons
- Fully styled FAQ section with plus icons
- Footer with email input and Get Started button
- 3-column footer links layout
- Google Fonts — Bebas Neue + Martel Sans
- Font Awesome 7 icons throughout

## Built With

- HTML5
- CSS3
- Font Awesome 7
- Google Fonts
